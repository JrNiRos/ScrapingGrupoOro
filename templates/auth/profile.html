Cómo obtener tu API Key

Puedes obtener una API Key en uno de los proveedores siguientes. Sigue estos pasos:

  1. Regístrate o inicia sesión en el proveedor (Serper o Google Cloud).
  2. Crea un proyecto (si es necesario) y habilita la API correspondiente (por ejemplo, Places API en Google).
  3. Ve a la sección de credenciales y crea una nueva API Key.
  4. Copia la clave y pégala en el campo "API Key" en esta página, luego guarda.
